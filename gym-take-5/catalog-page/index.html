Creating a CSS Knockout Text Effect

with Jeremy Osborn

In this video tutorial, you’ll learn how to create a knockout text effect with CSS. Knockout text is text that looks like it’s been cut out of a surface to reveal the background and can be give a unique design look for your website. In the past, this effect could only be created using an image editor such as Photoshop or Sketch.
